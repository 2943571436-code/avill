# 数字记忆挑战

Python 极简控制台小游戏（83 行），演示数据持久化。

## 启动

```bash
python memory_game.py
# 零依赖，Python 3.7+
```

## 玩法

随机数字逐位加长闪现 → 玩家凭记忆输入 → 正确继续，错误结束。

## 功能

| 功能 | 说明 |
|------|------|
| 🎮 游戏 | 数字序列逐位加长，闪现后消失，玩家输入 |
| 💾 持久化 | SQLite 单表存储最高分/游玩次数/累计分 |
| 🛡 校验 | 空输入/非数字/EOF/KeyboardInterrupt 全部捕获 |
| 📊 统计 | 主菜单实时显示最高分/平均分/游玩次数 |

## 数据存储

SQLite `memory_game.db`，单行记录累计统计。
