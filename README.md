# 并发猜数字游戏 —— 多线程稳定性测试系统

Python 多线程并发猜数字小游戏，对标 **系统并发稳定性测试** 场景。

## 快速启动

```bash
python guess_game.py
# Python 3.7+，无需安装任何第三方库
```

## 功能

| # | 功能 | 说明 |
|---|------|------|
| 1 | 🚀 完整对比测试 | 无锁→加锁两轮测试，自动生成对比报表 |
| 2 | 🔓 无锁模式 | 复现竞态条件、丢失更新、TOCTOU |
| 3 | 🔒 加锁模式 | 验证优化效果 |
| 4 | 📊 对比报表 | 报错率、数据一致性、冲突数、稳定性提升 |
| 5 | 🐛 冲突详情 | 查看无锁模式下的具体并发冲突日志 |
| 6 | 📋 历史会话 | 浏览所有测试记录 |

## 并发缺陷演示

| 缺陷 | 检测方式 |
|------|----------|
| 丢失更新 (Lost Update) | `实际记录数 ≠ 总尝试次数` |
| TOCTOU 竞态 | `重复获胜次数 > 1` |
| 脏读冲突 | 计数器读写间隙被其他线程修改 |

## 量化指标示例

```
▸ 并发报错率: 3.57% → 0% (改善 100%)
▸ 数据一致性: 16.1% → 100% (提升 83.9pp)
▸ 并发冲突数: N → 0 (减少 100%)
▸ 系统稳定性提升: 100%
```

## 数据存储

SQLite `concurrent_game.db` (WAL 模式)：
- `guess_records` — 猜测记录
- `conflict_logs` — 并发冲突日志
- `test_sessions` — 测试会话摘要
